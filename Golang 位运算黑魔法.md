在某些场景下，使用二进制数往往比使用其他数据结构更方便，性能也更好。通过对数据进行二进制位的运算，往往能更容易得到我们期望的结果。

1. 获得二进制数第一个1出现的位置。

```go
a := 8
fmt.Printf("%b\n", a) // 110
fmt.Printf("%b\n", a & (-a)) // 10
```

2. 将第一个1出现的位置清0

```go
a := 8
fmt.Printf("%b\n", a) // 110
fmt.Printf("%b\n", a & (a-1)) // 100
```

3. 对`2`的`n`次方取余

```bash
# 假设 y 等于 2 ^ n
x % y 等价于 x & (y - 1)
```
